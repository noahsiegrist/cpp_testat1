Falls Sie Visual Studio verwenden, dann verwenden Sie die Option "Open a local folder" und geben den Ordner "source" an.
Das CMake-File wird dann automatisch gestartet. Sobald CMake fertig ist, stehen zwei Build-Targets zur Verfügung:
- astro.exe
- astro_test.exe

Wählen Sie astro_test.exe zur Ausführung der Unit-Tests.
